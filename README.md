# San Alejo — Inventario de Objetos Guardados

Aplicación móvil desarrollada con **React Native + Expo Router + TypeScript** para gestionar contenedores (cajas, maletas, cajones, bolsas) y sus objetos, usando **SQLite local** sin backend.

## Tecnología usada

- React Native (Expo SDK 54)
- Expo Router (navegación)
- `expo-sqlite` (persistencia local SQLite)
- TypeScript

## Cómo compilar y ejecutar

1. Instalar dependencias:

   ```bash
   npm install
   ```

2. Iniciar la app:

   ```bash
   npx expo start
   ```

3. Abrir en emulador o dispositivo físico desde Expo (Android/iOS/Web).
